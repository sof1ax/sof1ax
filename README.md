## About Me
I studied Management Information Systems and worked across IT support and analytical roles, but learning to code last year changed everything. I found myself moving from backend to frontend to full-stack concepts, even picking up design along the way. For the first time, I could actually build the ideas I used to only imagine.

My curiosity used to feel like a weakness, but in tech, it became my advantage. Every new skill connects to the next, and every project teaches me something different.

If you’re someone who loves “too many things,” the truth is: you’re not lost — you’re built for tech.
